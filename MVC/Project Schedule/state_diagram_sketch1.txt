## Business Plan Skeleton ##

## Libraries Used (MVP)
- **React (v18)**: Core framework. Free, MIT license.
- **Tailwind CSS (v3)**: Styling for dark theme, cards, pulses. Free, MIT license.
- **Lucide React**: Icons. Free, ISC license.
- **React Router DOM**: Navigation. Free, MIT license.
- **wx-react-gantt**: SVAR React Gantt for interactive Gantt charts. Free, GPLv3 license. Install: `npm install wx-react-gantt`. Supports card-like tasks via `taskRenderer`, dependencies, subtasks, and Tailwind integration for pulses (e.g., `animate-heartbeatYellow`).
  - Alternative: Frappe Gantt (`npm install frappe-gantt`), MIT license, for simpler setups.
- **AG Grid Community**: Spreadsheet view. Free, MIT license.

## Backend: Python API and Database
- **Python API**: FastAPI for risk calculations (`/calculate-risk` endpoint). Mock for MVP (e.g., `riskLevel: 'medium'` based on duration). Full Monte Carlo/PDE/SDEs post-MVP. Deploy on Vercel (free).
- **Database**: MongoDB Atlas (free tier, 512MB) for user data (e.g., `{ userId, projects: [{ tasks: [{ riskLevel }] }] }`). Setup: Connect via `pymongo` in FastAPI. Scales to 800+ users (~$10/month upgrade).
